# 常见问题排查

## MQTT 连接失败

- 检查 `host/port` 是否正确。
- 验证用户名/密码。
- 确认 broker 可达。

## 控制权申请失败

- 遥控器端未授权。
- 当前已被其他用户占用。

## DRC 进入失败

- 未先申请控制权。
- broker 配置错误。
- `expire_time` 已过期。

## 心跳掉线

- 心跳线程未运行或被阻塞。
- 建议 5Hz 心跳。

## 直播失败

- RTMP URL 不可达。
- 镜头 video_id 或 video_index 错误。

## 飞点无响应

- DRC 未进入或控制权失效。
- 检查飞行模式与安全限制。
